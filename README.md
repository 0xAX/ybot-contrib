ybot-contrib
=============

ybot-contrib - is public repo for [Ybot](https://github.com/0xAX/Ybot) commutity plugins.
  
Usage
============

For plugin usage just copy in in your Ybot plugins directory.

Contribute
============

  * Fork main repository (https://github.com/0xAX/ybot-contrib).
  * Add your plugin.
  * Send pull request.
